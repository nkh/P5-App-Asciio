# Slides

**Navigation:** [Index](bindings_index.md) > [Modes](modes_operation.md) > Slides

**Prefix:** Press «S» to enter slide mode

**Exit:** Press «Escape» to return to normal mode

## Bindings

| binding     | Description           |
| ----------- | -------------         |
| «n»         | next slide            |
| «N»         | previous slide        |
| «g»         | first slide           |
| «s»         | run slideshow         |

##  Bindings while running a slideshow

| binding     | Description        |
| ----------- | -------------      |
| «n»         | next slide         |
| «N»         | previous slide     |
| «s»         | slower             |
| «f»         | faster             |
| «p»         | pause              |
