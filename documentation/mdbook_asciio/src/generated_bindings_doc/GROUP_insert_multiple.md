# insert_multiple

Navigation:

Insert multilpe elements at once

- Shortcut: «insert_multiple»

| Operation                    | Binding | Description |
| -                            | -       | -           |
| Add multiple boxes           | «000-b» |             |
| Add multiple box connected   | «00S-B» |             |
| Add multiple texts           | «000-t» |             |
| Add multiple texts connected | «00S-T» |             |
