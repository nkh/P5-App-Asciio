# grouping ->

Navigation:
	- [grouping ->](GROUP_grouping___.md)

Group elements and control their Z position

- Shortcut: «000-g»

| Operation                   | Binding | Description |
| -                           | -       | -           |
| Group                       | «000-g» |             |
| Ungroup                     | «000-u» |             |
| Move to the front           | «000-f» |             |
| Move to the back            | «000-b» |             |
| Temporary move to the front | «00S-F» |             |
