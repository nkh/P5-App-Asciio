# Find

Find mode searches for text within elements and navigates through results.

| Operation     | Binding   |
| -----------   | --------- |
| clear search  | «Ctl+l»   |
| start search  | «\*»      |
| find next     | «n»       |
| find previous | «N»       |

# Find Mode
