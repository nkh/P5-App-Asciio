# Scripting examples

## from_command_line

the directory contains examples you run on the command line

- you can run asciio script from anywhere 
- asciio doesn't need to be started
- the script suse the simplified API

## from_within_asciio

the examples must be run from asciio

- open the script via a file browser (binding ":!")
- open the script via option '-s'
     asciio -s ./asciio_script (note the '.' for files in the current directory)

## web_interface

asciio must be running

the examples will connect to asciio web interface and send commands to it

the results are visible in the asciio UI
